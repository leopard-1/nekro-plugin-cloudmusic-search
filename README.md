# Cloudmusic-Search

> 以nekro-agent为框架的网易云音乐搜索工具

具体功能
1.发送音乐JSON卡片（或语音）
2.支持检索音乐作曲家，对应专辑，封面URL等
3.下载并发送音频文件
4.支持VIP音乐

## 如何使用？
由于该插件不靠指令触发，您需要向AI提供相关信息以检索
### 例如：搜索音乐：稻香 -周杰伦
之后，您会获得一个音乐搜索列表（默认15个/页，可自行修改）。背景默认为纯色，可以使用图床直链自定义更改，如https://www.img520.com/TeNMli.jpg
 > 建议尺寸:800*800
选择对应的序号即可获取，可以额外要求发送语音，下载，查询信息等
